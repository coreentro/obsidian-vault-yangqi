

## 一、配制稀盐酸

**5 mL 浓盐酸 + 纯水定容至 50 mL**，搅匀备用

---

## 二、配制显色剂

1. 称取 **4 g 对二甲氨基苯甲醛**
2. 加入 **20 mL 稀盐酸**
3. 加入 **200 mL 无水乙醇**
4. 搅拌至完全溶解，棕色瓶避光保存

---

## 三、配制水合肼标准液

**第①步：1000 mg/L 储备液（50 mL）** **60.7 μL 80%水合肼 + 49.939 mL 纯水**

**第②步：10 mg/L 工作液（10 mL，标准曲线用）** **100 μL 储备液 + 9.900 mL 纯水**

**第③步：5 mg/L 降解液（50 mL）** **250 μL 储备液 + 49.750 mL 纯水**

---

## 五、催化降解实验

1. 取50 mL、5 mg/L降解液，放入反应装置
2. 加入 **20 mg 催化剂**
3. 开启搅拌/光照，记为 **0 h**
4. 每隔1小时取样一次，共取6次（0、1、2、3、4、5 h）
5. 每次取 **0.5 mL** 反应液，立即针管过滤去除催化剂

| 组分     | 体积       |
| ------ | -------- |
| 样品上清液  | 0.2 mL   |
| 纯水     | 2.8 mL   |
| 稀盐酸    | 0.5 mL   |
| 显色剂    | 0.5 mL   |
| **总计** | **4 mL** |
1. 0h、1h、2h、3h、4h、5h 的上清液**同时**各取0.2 mL
2. 同时加纯水3.8 mL
3. 同时加稀盐酸0.5 mL
4. 同时加显色剂0.5 mL
5. 盖塞摇匀避光，**统一计时30分钟**
6. 时间到后统一测A值
