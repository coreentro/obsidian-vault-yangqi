# 水介导反应

## 1. 定义

水介导反应指的是水不仅仅作为普通溶剂存在，还可能通过氢键、酸碱作用、疏水效应或界面效应等方式促进反应进行。

## 2. 在本文献中的意义

本文献中，水的作用可能包括：

1. 作为绿色溶剂；
2. 提供反应介质；
3. 作为弱 Brønsted 碱，帮助活泼亚甲基化合物脱质子；
4. 促进醛和活泼亚甲基化合物在界面或悬浊体系中反应。

## 3. 为什么水值得关注？

水的优点：

- 便宜；
- 安全；
- 无毒或低毒；
- 环境友好；
- 易获得；
- 后处理相对简单。

水的缺点：

- 很多有机物在水中溶解性差；
- 有些试剂或催化剂可能被水破坏；
- 有些反应在水中难以控制。

## 4. 本文献的发现

作者发现，在苯甲醛和丙二腈的模型反应中，水比许多常见有机溶剂效果更好。

不同来源的水，例如 HPLC 水、瓶装水、去离子水，都能促进反应。

这说明反应不是由自来水中的痕量金属杂质导致的。

## 5. 溶剂筛选结论

| 溶剂 | 结果 |
|---|---|
| 水 | 最好 |
| 甲醇 | 可以 |
| 乙醇 | 一般 |
| DMF、DMSO | 有一定效果但不理想 |
| 二氯甲烷、乙酸乙酯、乙腈、甲苯 | 基本不反应 |

## 6. 注意事项

水介导不等于所有水中反应都能成功。

该方法对底物有要求：

- 醛类更适合；
- 丙二腈等强活泼亚甲基化合物更适合；
- 酮类不适合；
- 溶解性差的底物可能需要 H₂O/MeOH 混合溶剂。
