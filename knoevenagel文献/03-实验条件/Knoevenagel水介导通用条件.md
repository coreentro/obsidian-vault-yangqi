# Knoevenagel 水介导通用条件

## 1. 反应类型

Knoevenagel 缩合。

## 2. 适用范围

适用于：

- 醛类底物；
- 丙二腈；
- 氰乙酸乙酯；
- 部分巴比妥酸类活泼亚甲基化合物。

不适合或效果差：

- 酮类，如苯乙酮、环己酮；
- 丙二酸二乙酯；
- 苯乙腈；
- 乙酰丙酮；
- 3-吗啉丙腈。

## 3. 通用条件

- 醛：1.00 mmol，1.00 equiv.
- 活泼亚甲基化合物：1.00 mmol，1.00 equiv.
- 溶剂：H₂O 2 mL
- 如果底物溶解性差：H₂O/MeOH = 1/1，总体积 2 mL
- 温度：室温
- 时间：20 min–18 h
- 监测：TLC
- 装置：玻璃小瓶，搅拌
- 后处理：旋干溶剂
- 必要时：柱层析纯化

## 4. 模型反应

苯甲醛 + 丙二腈 → 2-benzylidenemalononitrile

## 5. 溶剂筛选结果

| 溶剂 | 表现 |
|---|---|
| H₂O | 最佳 |
| 甲醇 | 可以 |
| 乙醇 | 一般 |
| DMF、DMSO | 有一定效果但不理想 |
| 二氯甲烷、乙酸乙酯、乙腈、甲苯 | 基本不反应 |
| 无溶剂 | 不反应 |

## 6. 操作理解

这个方法的操作重点不是复杂仪器，而是：

1. 底物是否适合；
2. 水或水/甲醇体系是否能让底物接触；
3. TLC 是否显示反应完成；
4. 反应后是否能直接旋干得到较纯产物。

## 7. 风险点

- 底物溶解性差时可能影响反应；
- 对酮类不适用；
- 对低活性亚甲基底物不适用；
- 反应时间差异较大，需要 TLC 监测；
- 如果粗产物不纯，仍然需要柱层析。
