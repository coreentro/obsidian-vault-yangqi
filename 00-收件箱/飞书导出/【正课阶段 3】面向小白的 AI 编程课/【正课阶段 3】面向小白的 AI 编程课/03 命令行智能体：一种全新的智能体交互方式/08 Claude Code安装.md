# 写在前面



你好，我是大圣



这节课我们开始正式进入到Claude Code的安装



你可能会问，不就是装个软件吗，为什么还需要专门写一系列教程？原因有三：



**第一，安装过程对新手不太友好。** Claude Code 是一个命令行工具，所有操作都需要在命令行中完成。而且它的安装依赖一些前置工具，对编程小白来说会有一定门槛



**第二，登录验证需要绕过。** 我们前面提到过，Claude Code 本质上是一个编程工具，要让它真正工作，需要接入一个大模型。



但 Claude 目前对国内用户并不友好，国内账号几乎无法正常使用，即便注册成功也大概率会被封号。因此，在选择模型和登录这一步，我们需要用一些魔改的方法来绕过官方的验证



**第三，我们需要额外安装一个工具。** Claude Code 接入模型时需要手动修改配置文件，操作比较繁琐。而通过这个工具，不仅可以简化配置流程，还能让 Claude Code 突破官方模型的限制，接入任意第三方大模型



所以接下来我会写多篇教程，来让大家把这个安装流程完完全全地走一遍



每一篇教程只解决一个问题，让大家不仅能实操，而且了解它背后的原理



# 几点约定



在这节课之前，我已经有一篇教程帮助上千名学员安装Claude Code



在安装过程中，我看到了这样或者那样的问题。为了帮助大家更好地完成这件事儿，我们做如下几个约定



1. **安装也是学习的一部分**。我们的目的不仅仅是安装成功，而是借助这个过程获得一些新的编程知识。请大家带着学习的心态来安装



1. **每个人的电脑环境、网络环境都不一样**。我会尽最大的努力考虑到所有情况，但你可能会遇到这样或那样的问题，请具备一定的折腾精神



1. **请务必全程与你的AI协作去做这件事。遇到问题优先问AI**



1. 我们在这个过程中会借助**Kimi Code**来辅助我们完成一些事情



1. **不要跳着看教程，每一个字、每一个视频都要看完再来提问**
