# 写在前面



你好，我是大圣



我们既然要学命令行智能体，那首先我们得有命令行



所以这节课的目的就是在你的苹果电脑或者Windows电脑上能够正确打开命令行



# 一、安装命令行工具



命令行工具，简单来说就是上个时代程序员写代码用的界面：通过手动输入命令来操作电脑



Claude Code 、Kimi Code的原生运行方式就是命令行，所以我们首先需要确保你的电脑上有一个可用的命令行工具



这里分 Mac 和 Windows 两种情况



## 1）Windows电脑的命令行工具



Windows 系统上的命令行工具主要有两种：一种是最原始的 CMD，另一种是功能更强大的 **PowerShell**



为了保持操作统一，**我们全程使用 PowerShell**。



从 Windows 7 到 Windows 11，系统都会自带 PowerShell，只是版本略有不同



> *⚠️* ***特别提醒：*** *如果你的系统是 Windows 7，后续操作中可能会遇到一些兼容性问题。因为我手头只有 Win10 系统，无法对 Win7 逐一验证。如果你在操作过程中碰到异常，可能是系统版本的问题*
