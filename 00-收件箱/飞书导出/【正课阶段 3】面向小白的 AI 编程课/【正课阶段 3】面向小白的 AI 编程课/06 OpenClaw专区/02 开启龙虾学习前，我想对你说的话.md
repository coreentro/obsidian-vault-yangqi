# 开启龙虾学习前，我想对你说的话

<callout emoji="✨">
最近为了给大家写龙虾教程，一直在看网上的各种资料、各种案例，真的没啥用
首先跟大家声明，我下面写的教程纯粹是为了蹭这波热点，纯粹是为了满足社群里面大家对龙虾的好奇心
从我自身的角度而言，Claude Code 比龙虾好用得多
当然，如果你可以自己买服务器，或者自己买电脑，自己去部署一个龙虾，用最强的模型，那它应该有很多的玩法
但我认为这对于普通人门槛太高了，而且你要探索玩法，说白了还是探索属于自己的工作流
因为我的风格是面向小白，一步一步地去掌握这个工具。
所以一开始，无论是部署方式还是一些教程我都会用最简单的方式，简单到它可能只有学习价值，而没有实用价值
但我有自己的愿景：
**我期望通过Claude Code的教程和OpenClaw的教程，让大家把AI时代的基础知识全部掌握**
**后面再出现什么小螃蟹、小乌龟这样的工具的时候，我们真的能够自己去学习**
这是我在写小龙虾教程的时候，核心要考虑的点
也分享给各位，希望大家带着这样的目标去学习
---------
写在最后
**如果你想干一些实事，我建议你把 Claude Code 真的去玩明白，去网上找各种各样的教程，去融入到自己的实际工作中，去打磨一个又一个的Skill**
**如果你真的对龙虾心痒难耐，特别好奇，那你继续往后看课，我不会划水，我会认真做龙虾的教程。但需要时间**
</callout>
