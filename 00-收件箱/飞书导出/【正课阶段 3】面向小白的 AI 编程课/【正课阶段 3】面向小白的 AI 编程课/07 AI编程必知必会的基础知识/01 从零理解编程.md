# 写在前面



在讲AI编程之前，我们必须要先对编程这个概念有一定的了解。



这一章，我们就用通俗易懂的语言来讲解下编程是什么？



# 为什么要编程



在了解任何一个概念之前，我习惯先了解Why，然后再来了解What



因此在我们深入了解**编程是什么**之前，我们先来思考一个问题：**为什么需要编程**？



## 1、编程的诞生



1. **计算机的发明：**人类发明了计算机，这是一个强大的工具，能够进行快速计算和数据处理。我们很快发现，计算机可以帮助我们解决各种复杂的问题。



1. **使用计算机的需求：**有了这个强大的工具，自然而然地，我们想要充分利用它。但是问题来了：我们该如何告诉计算机我们想要它做什么呢？



1. **编程概念的诞生：**为了解决这个问题，**我们创造了编程这个概念**。编程就像是人类和计算机之间的翻译官，将人类的意图转换成计算机能够理解和执行的指令。



## 2、编程的核心目的



所以，编程的核心目的很简单：**指挥计算机按照人类的意愿行事，解决人类的特定问题**



这个目的包含了几个重要的方面：



1. **沟通**：编程是人类与计算机沟通的方式。
2. **控制**：通过编程，我们可以控制计算机的行为。
3. **自动化**：通过编程，我们可以让计算机自动完成重复性的任务。



---



**理解编程的目的是解决问题这个点很关键**：



1. 它帮助我们聚焦于问题解决，而不是纠结于技术细节。
2. 它提醒我们，编程是一种工具，而不是目的本身。
3. 它鼓励我们思考如何更好地利用计算机来解决实际问题。



# 什么是编程



理解了为什么需要编程之后，我们接下来就需要了解编程是什么？



对于编程是什么？我给出我的三个理解：



1. **编程给计算机下指令的过程：**编程本质上是告诉计算机如何一步步完成任务。就像你告诉朋友如何从你家到超市，需要详细、准确的指示。



1. **编程一种沟通的方式：**编程是人类与计算机之间的沟通。我们使用特定的"语言"（编程语言）来表达我们的想法和意图。



1. **编程是问题解决的工具：**编程不仅仅是写代码，更是一种思考和解决问题的方法。它培养逻辑思维和创造性解决问题的能力。



# 如何编程



## 1、编程语言：与计算机沟通的桥梁



前面我们讲到，编程就是给计算机下达指令的过程。



那么，计算机该如何听懂我们的指令呢？这就引出了一个关键概念：**编程语言。**



编程语言就像人类语言一样，种类繁多。常见的有：

- Python
- Java
- C++
- JavaScript
- 等等



你可能会问："为什么会有这么多编程语言呢？"，因为每种语言都有其特定的应用场景和优缺点。



**但对于我们本次教程的目标人群来说，特别是编程小白，建议直接选择Python。**



无需纠结于语言的选择，重要的是开始学习。



## 2、编程平台**IDE**：指令的传递者



学会了编程语言后，你可能会问："我该怎么把我的指令下达给计算机呢？"



我总不能对着计算机念我用Python写的代码吧？



这个问题引出了另一个重要概念：**编程平台，又被称为IDE**



**编程平台的作用是将你用编程语言写的代码传递给计算机，让计算机执行你的指令**。



为什么需要编程平台？这里有个有趣的事实：



<callout emoji="⚽">
计算机实际上只能理解二进制指令，也就是一串0和1的组合。这种二进制指令对人类来说就像天书一样难懂。如果要直接用二进制编程，学习难度会大大增加。
</callout>



我们日常使用的编程语言，如Python，被称为高级编程语言。



"高级"的意思是它更接近人类的自然语言。



编程平台的一个重要功能就是将这些高级语言转换成计算机能理解的二进制指令。



## 3、编程语言、编程平台和计算机的关系



让我们用一个开车的例子来说明这三者的关系：



1. 你买了一辆问界M5的汽车，想用它解决出行问题。（汽车 = 计算机）
2. 为了操作汽车，你去驾校学习。（学习驾驶 = 学习编程语言）

   - 你学会了：向左打方向盘让车向左转
   - 你学会了：向右打方向盘让车向右转
3. 这里的"向左打方向盘"、"向右打方向盘"就相当于编程语言中的指令。
4. 而方向盘则相当于编程平台。它让你无需了解汽车的底层机械细节，就能把指令传达给汽车。



那么编程平台有哪些呢？我们又该如何选择？



前面讲过，对于AI时代的编程者而言，我们的目的并不是成为程序员。



所以我们并不需要像专业程序员那样使用复杂的开发工具。



所以对于编程平台，你可以这么理解：



- 如果你使用Coze的代码节点，那么Coze就是你的编程平台。
- 如果你使用Cursor进行编程，那么Cursor就是你的编程平台
- 如果你使用字节的MarsCode进行编程，那么MarsCode就是你的编程平台。



这些平台为我们提供了一个便捷的环境，让我们能够专注学习编程的核心概念，而不用过多地担心技术细节。



# 写在最后



本文我们探讨了编程的本质、目的和基本方法：



1. **编程的目的**是指挥计算机按照人类的意愿行事，解决特定问题。
2. **编程的本质**是一种与计算机沟通的方式，一种解决问题的工具。
3. **编程语言**是我们与计算机沟通的桥梁，而**编程平台**则帮助我们更轻松地传达指令。



理解这些概念不仅能帮助我们更好地利用AI工具，还能培养我们的逻辑思维和问题解决能力。



在AI时代，编程不再仅仅是程序员的专属技能，而是每个人都应该掌握的基本素养。



但请记住，在AI时代，普通人学习编程不是为了成为专业的程序员。



而是为了更好地理解和利用技术，实现我们的创意，解决实际问题。



所以大家在后面的教程中千万不要陷入技术的细节之中，而是要了解更本质点。



随着AI技术的不断进步，编程的方式可能会发生变化，但其核心：**解决问题的思维方式将始终保持重要**。
