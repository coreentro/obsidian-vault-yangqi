# 写在前面



你好，我是大圣



我们讲到扣子的愿景是打造一个真正的数字员工



而一个真正的数字员工，他应该有自己的日程排期，有自己趁手的工具，甚至还有自己的身份工牌



扣子用三个模块来实现这件事：日程让它知道什么时候干，设备让它有手去干，邮箱让它有身份去干



![](https://feishu.cn/file/G5XcbIPM4obyXfxrkbPc7BzCnhh)



# 一、日程：扣子的排班表



日程的本质不是一个日历功能，而是**触发器，类似于闹钟**



它让扣子从被动等你说话变成了主动按时干活。这是扣子从工具走向数字员工的关键一步



![](https://feishu.cn/file/QaWXb4x2VobR8IxjrIscFJvXneb)



你用自然语言创建好日程之后，它会出现在日程这个导航里面



![](https://feishu.cn/file/MPw0bvkARopuoRx0DUWcck9Onrc)



![](https://feishu.cn/file/YBZfb4UBioewvSxLQVUcTbZMnog)



# 二、设备：扣子的手和眼



云设备解决的就是这个问题：**给扣子一双手和一双眼，让它能操作真实的软件和网页**



其实，Agent的发展本质就是模型能力的发展，然后可以准确地操作更多的设备、使用更多的工具



PS：我要提前说一嘴，AI还在发展中，虽然扣子配备了设备，但使用的体验还不是那么好



## 2.1 两种设备



![](https://feishu.cn/file/ZGWsbUCPyo5N5txvQBtcGUXenwd)



**云手机**：一台真实的Android 13手机，运行在云端。



扣子可以在上面下载APP、打开应用、滑动页面、点击操作。你可以实时看到它的屏幕



能做什么：刷小红书查笔记、用大众点评找餐厅、在携程上筛酒店、打开微博看热搜、回复抖音私信……所有你在手机上能做的事，扣子都能代劳



**云电脑**：一台Ubuntu系统的云端电脑。扣子可以在上面运行代码、浏览网页、安装软件、处理文件



能做什么：执行Python脚本、抓取网页数据、截取网页长图、填写在线表单、做自动化测试



![](https://feishu.cn/file/UaNVbynBeolpL5xGywScuaoEntf)

## 2.2 两个重要的注意事项



**账号隔离**：强烈建议为扣子注册专属的APP账号



不要让它登录你的私人大号。高频自动化操作可能触发第三方平台的风控，导致封号



这不是扣子的问题，是第三方平台的规则



**后台运行**：云设备的操作在后台进行，不占用你的对话窗口。你派完活可以继续和扣子聊别的



## 2.3 开通条件



云设备不是所有用户都能用。需要个人高阶版及以上的套餐。每个用户最多1台云手机+1台云电脑

# 三、邮箱：扣子的工牌



![](https://feishu.cn/file/KQHAbU11AoSUQyxrX2fcEIJbnPb)



## 3.1 它解决什么问题



日程让扣子知道什么时候干活，设备让它有手去干活。但它还缺一样东西：**身份**



没有身份，它就没法注册第三方平台的账号、没法收发邮件、没法和别人通信。



它依然是一个"关在对话框里的AI"



扣子邮箱解决的就是这个问题。每个扣子都可以注册一个专属邮箱地址（格式是 xxx@coze.email），这就是它在数字世界里的工牌



## 3.2 邮箱能做什么



- **代你发邮件**：在对话里说"给 zhangsan@example.com 发邮件，约他今晚7点吃饭"，扣子就用自己的邮箱帮你发出去



- **自动处理收件**：扣子会定期检查新邮件，根据内容判断怎么处理。你可以设定规则："来自bytedance.com的邮件一定要提醒我"、"平台状态更新直接标记已读"、"其他Agent发来的邮件你先自行处理"



- **垃圾邮件拦截**：钓鱼邮件、带毒附件自动拦截。安全系统处理，你不需要操心



- **和其他Agent通信**：扣子可以用邮箱和其他人的Agent互发邮件、交笔友。当Agent之间能直接通信，就意味着它们可以协作完成跨用户的任务



## 3.3 邮箱的本质



邮箱表面上是一个通信工具，但它的本质是**身份基础设施**



有了邮箱，扣子可以注册账号、验证身份、接收通知、建立社交关系。



它从一个"没有名字的工具"变成了一个"有身份的个体"



这也是扣子在Agent World里能社交、能学习、能探索的前提，没有身份，连门都进不了



# 四、三者怎么配合



单独看每一个模块，它们只是三个功能。但合在一起看，它们构成了一个完整的"自主执行闭环"：



**日程触发任务 → 设备执行操作 → 邮箱提供身份 → 结果自动交付**



举一个串联的例子：



你告诉扣子："每天早上9点，用云手机打开微博，收集今日热搜前10条，整理成早报，发邮件给我。"



扣子会做什么？

1. **日程**：创建一个每日9:00的循环任务
2. **设备**：到点自动启动云手机，打开微博，浏览热搜，截图记录
3. **邮箱**：整理完毕后，用扣子邮箱把早报发到你的个人邮箱
4. 你早上打开邮箱，热搜早报已经躺在那里了



整个过程你不需要在场。这就是自主执行的完整链路
