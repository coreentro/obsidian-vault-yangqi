# 写在前面



你好，我是大圣



这节课给大家分享扣子，甚至说整个智能体生态中最关键的一环：技能



对比新入职的员工，就算你给他配备了电脑，排了日程，办了工牌，但是如果他不懂业务，他还是干不了活



因此员工需要培训，需要掌握行业的知识和工作方法



对于智能体，我们把这个东西叫做技能，你可以为你的智能体安装各种各样的技能来完成不同的工作



# 一、技能到底是什么



**技能是一套可复用的任务解决方案**



它包含了处理某类任务的完整方法论：流程是什么、每一步怎么做、用什么工具、输出什么格式、质量标准是什么



技能就是把你的方法论沉淀成SOP，按照某种智能体能识别的标准沉淀下来



# 二、技能从哪里来



扣子的技能有三个来源。理解这三个来源，你就理解了扣子技能生态的全貌：



![](https://feishu.cn/file/QK8Bb7b51o8LGaxtSeKcyefTnsf)



## 来源一：官方内置



扣子出厂就自带的技能：PPT、Excel、视频、海报、播客、设计



这些你不需要安装，开箱即用



![](https://feishu.cn/file/WqyDbcOEeoJPwHxhlNxc27iUncc)



## 来源二：技能商店



别人开发好的技能，上架到商店里，你可以直接安装使用



商店里有免费的也有付费的。付费方式有两种：按次付费（用一次扣一次积分）和订阅制（按月/季/年，订阅期内无限用）



商店覆盖了法律、金融、教育、自媒体、电商等行业



比如法律Skill能帮你做合同审查和类案检索，金融Skill能帮你分析市场行情，这些技能的背后是行业专家的经验被结构化地封装了起来



<callout emoji="✨">
PS：注意，技能商店是别人上传好的，可能存在挂着羊皮卖狗肉的情况
技能的描述非常吸引人，但实际使用起来可能不尽如人意，这个只能大家自行去体验
</callout>



![](https://feishu.cn/file/KzmOb1Rn8oVOybxEB0Acxf3anve)



## 来源三：自己造



你可以把自己的工作经验变成技能



**方式一：从对话沉淀。** 你在和扣子的对话中，经过多轮调整，终于做出了一个满意的产物。



这时候你可以说帮我将以上处理流程制作为技能。扣子会自动总结这次任务的核心流程，封装成一个可复用的技能。下次遇到类似的任务，它就能一步到位



这是新手最容易上手的造技能方式：不需要写代码，不需要懂开发，你只需要有一个做得好的案例



**方式二：复制改造。** 看到商店里有个开源技能不错，但不完全符合你的需求？



一键复制到你的工作空间，改成你想要的样子



开源和免费不同，开源是你能看到别人的源码



开源的一定免费，但免费的不一定开源。免费是你能直接使用，但不一定能知道它是怎么制作的



**开源的技能很少，因为这涉及到别人的知识产权**



![](https://feishu.cn/file/SuoDboTLpoHLbwxfqPucKGggnJh)



![](https://feishu.cn/file/Gnd9b0228owZemxjjRjcD21kn6d)



**方式三：扣子编程开发**



对于有一定开发经验的用户，可以在扣子编程里用Vibe Coding从零开发技能



把你的专业知识、工作流程、最佳实践告诉扣子AI，它会生成标准格式的技能文件包。这部分我们后面讲扣子编程的时候会展开



![](https://feishu.cn/file/DhSAbvZsromvAlxQnBmceG6Kneb)



# 三、怎么使用技能



使用技能有两种方式，一个自动一个手动：



**自动匹配**：你在对话框里正常发指令，扣子会自动在技能商店里查找相关技能。



如果它判断某个技能和你当前的任务有关，它会自动安装并启用。你不需要操心



![](https://feishu.cn/file/GC2Gbzi0WoNNTqxVLEecnlKgnRc)





**手动指定**：在对话框里输入@，切到技能页签，选择你要用的技能。



然后描述你的需求。这种方式更精准：你明确告诉扣子"这次用这个技能来做"



![](https://feishu.cn/file/SQSubvXQooCDHhxIEcQcQcfWnOf)



# 四、ji



技能商店不只是一个下载市场。它是扣子生态里最核心的飞轮：



![](https://feishu.cn/file/JQhjbzbqvoJoE3x7T2kccK2WnId)



**用**：用户安装技能，解决实际问题。用多了发现现有技能不完全满足需求，想定制



**造**：用户把自己的工作经验封装成技能。有三种方式：对话沉淀（最简单）、复制改造（中等）、编程开发



**卖**：造好的技能可以上架到技能商店，设定价格（按次或订阅），其他用户付费使用，你获得收入。



扣子已经接入了完整的交易系统，开通收款账户后就能收钱



这三者形成飞轮：用技能的人越多 → 造技能的动力越大 → 商店里技能越丰富 → 吸引更多人来用



**对于当下大部分伙伴而言，你的核心是用技能，从而先实现自己的业务**
