# 写在前面



你好，我是大圣



这节课给大家讲一下和扣子对话的基本操作



# 一、配置你的Agent



首先你可以对你的Agent进行最基本的配置，你可以给他起一个名字



![](https://feishu.cn/file/CPCCbHk57oF7tMxcglHc7TLMnyf)



你可以设置跟你的扣子沟通的渠道，目前扣子支持飞书和微信两个渠道



![](https://feishu.cn/file/TYddbH2cZo7BhmxW0Kqcai8kn5f)



你在网页端交代过的事情，在飞书里问它也记得。记忆是共享的



**但对话是独立的：网页端、飞书、微信各自有各自的对话历史**



这是有意为之的：不同渠道的对话场景不同，混在一起反而乱



飞书渠道值得多说一句。它不只是"在飞书里聊天"这么简单。



绑定飞书之后，扣子可以直接操作飞书的功能：创建文档、管理多维表格、查日程、发消息、创建任务。等于说你在飞书里有了一个24小时在线的全能助手



# 二、对话里能带什么进去



与AI沟通的核心技巧之一就是提供充足的上下文



这里核心给大家讲的是扣子的对话里面能带进哪些上下文



## 1.1 带文件进对话



点对话框旁边的"+"，可以上传文件。图片、文档、表格、PDF都行



典型场景：你上传一份销售数据表，然后说帮我分析Q1增长最快的品类，做个图表



扣子会读取文件内容，结合你的指令产出结果



你也可以从文件系统里引用已有的文件：不需要重新上传，直接在文件列表里选"引用到对话"



## 1.2 用@调用技能和收藏夹



在对话框里输入@，会弹出一个选择器，可以选择两种东西：



**技能**：比如@PPT、@Excel、@视频、@设计。选中后扣子会在这次任务中启用对应的专业技能



**收藏夹标签**：比如你之前收藏了一堆"英伟达"相关的资料并打了标签，这时候@英伟达，扣子会把这个标签下所有的资料都加载进上下文，然后基于这些资料回答你的问题



**一个实用的组合技**：上传文件 + @技能 + 文字指令。比如：上传一份PDF报告 → @Excel → "把这份报告里的关键数据整理成Excel表格，按季度分类"。三者叠加，产出质量远高于纯文字描述



# 三、扣子的对话上下文设计



扣子的对话是有上下文的。你在同一个话题里说过的话，它都记得



但截止到2026年4月份为止，我还是很不习惯它的主对话不能清除



![](https://feishu.cn/file/EgY4b3FBVo6KIxx4AZ5cqTxXnHd)



我们之前的习惯是开一个对话，聊完当前话题之后再新开一个对话。每一个对话只聊一个话题，防止污染上下文



但扣子我不知道它当前为什么这么设计，可能有它的思考



幸运的是他给我们提供了一个新建话题的能力，就是我们依然拥有一个对话了解一个聊一个话题的方案
