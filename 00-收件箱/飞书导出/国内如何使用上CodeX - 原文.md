# 写在前面



你好，我是大圣



你可能会疑惑，你不是刚讲完Claude Code吗？CodeX又是个什么鬼？



我们前面讲过这个图，Claude Code是Anthropic开发的智能体框架



CodeX则是OpenAI开发的通用智能体框架



那为什么我们已经安装了Claude Code，却还要讲CodeX？它们之间有什么不同？这节课我全部给你讲明白

# Claude Code的困境



我前面讲过，无论是Claude Code、Kimi Code、WorkBuddy，还是我们这节课讲的CodeX



他们本身都只是一个执行架构，相当于人的身体



而真正决定你的Claude Code是否好用，**本质在于它的AI大脑，也就是你给Claude Code配置的模型**



![](https://feishu.cn/file/MIa0b3uH4oVsUGxcFFTcPWZRnhg)



那在国内使用Claude Code有一个困境，就是我们可以下载好Claude Code，但如果我们想使用Claude这个最顶尖的模型，我们得找各种中转站



而各种中转站良莠不齐，费用极高，最重要的是还可能掺水



单纯从性价比上讲，一定是Claude的官方订阅性价比最高。无论是20美金、100美金还是200美金



**那你立马会问了，大圣，官方订阅性价比这么高，为什么你不给我们讲？**



# 为什么我们不搞Claude官方订阅？



因为Claude的风控非常严重



因为一些大家都懂的原因，国内访问海外网站本来就有限制



虽然我们日常有一些工具能够突破这个限制，但因为Calude风控严重，它还需要一些额外的技巧



这些额外的技巧包括，但不限于：**住宅IP、链式代理、代充、手机接码**



PS：如果你感兴趣，可以去Youtube上搜这些关键词，了解其原理



所以不是不给大家讲Claude官方订阅，而是它的限制太多，而大多数人不喜欢折腾，这个话题也比较敏感



# CodeX解决什么问题



那CodeX又能解决我们什么问题？大圣，你为什么要讲CodeX？



这里先给大家科普一个前提：当前做AI编程，做规划最好的两个大模型：**GPT 5.5 和 Claude Opus 4.7 & 4.6**



CodeX是OpenAI推出的工具



我们推崇CodeX，并不是CodeX比Claude Code好，它们两个是一类工具，不分高下



所以现在各种自媒体上讲的：**我们要使用CodeX，本质是我们要使用具有GPT5.5大脑的CodeX**



而是CodeX可以解决我们三个问题



**第一：买了ChatGPT 20美金的官方订阅，我们就能在CodeX中使用GPT5.5**



CodeX的账号体系、会员体系和GPT的网页端是共享的



所以你拥有ChatGPT 20美金的官方订阅，你就可以在CodeX中使用GPT-5.5的模型



而在GPT5.5加持下的CodeX好用程度，仅次于Claude Opus模型加持下的Claude Code



甚至在某些方面，它要比Claude模型更好用



**第二：Claude 20美金的官方订阅消耗起来特别快，而ChatGPT 20美金的官方订阅，从体感上来讲，额度是Claude Code 20美金的5倍不止**



**第三：GPT的风控程度目前而言比Claude要轻一些**



截止到2026年5月6号，**使用GPT和CodeX可以不用住宅IP，他们风控相对较低**



但使用Claude还是强烈建议上住宅IP



后面我再给大家写住宅IP的教程，如果你手里面有现成的网络工具，理论上来说是可以用GPT的



**第四：GPT的Image 2.0，目前，全球最强的图像模型**



GPT的Image 2是全球最强的图像模型。CodeX里面可以直接使用



![](https://feishu.cn/file/QndGbBOj6otBBSxMDKVcCdqrnCt)



# 四个优先级



综上所述，在综合考虑成本的情况下



第一：**拥有1个GPT 20美金官方订阅的CodeX**



第二：**使用Claude Code加国内的模型，比如GLM5.1 或者 MinMax2.7**



第三：**使用Claude Code加Claude的20美金官方订阅**



**第四：使用Claude Code加Claude的中转API**



如果你用量大、又能解决网络问题，那使用Claude Code的100美金、200美金官方订阅一定是划算的



关于Claude的官方订阅问题，我们不在这节课讨论



---



注意这个优先级它是一种综合考虑成本的情况下得到的，不要抬杠，你可以根据自己的需要去选择



他并不一定适用每个人



比如我使用的是GPT 20美金官方订阅 + CodeX、Claude官方订阅的100美金会员 + Claude Code

## **第二个问题：充值问题**



海外工具不能使用国内银行卡充值，必须使用海外信用卡



于是就产生了代充这个业务



代充有很多种方式，比如苹果的礼品卡，比如一些利用网站的漏洞



像ChatGPT就在2026年的5月份之前，因为它的漏洞，你只需要80块钱，就可以享受一个月20美金的服务



但是官方发现了这个漏洞，并且进行了清算，很多非法代充的会员都调回了免费版本



**闲鱼全是这种代充服务：你可以搜索拼车、代充关键词**

## 第三个问题：账号问题



要使用CodeX和GPT，你必须有一个账号



如果你有自己的账号，你可以找别人代充



如果你搞不定自己的账号，那你直接可以买成品号



我上面推荐的那个微信，它也提供成品号的服务，具体价格自行询问



## 第四个问题：手机接码



CodeX在2026年4月下旬需要验证手机号



这个手机号不能是国内的手机号，所以它需要使用海外的手机号



而我们几乎没有过海外的手机号，于是就诞生了一个业务叫做接码平台



在这个平台上，你可以挑一个海外手机号，然后接收验证码，支付一定的费用



我上面推荐的那个人，他卖的成品号已经经过手机号的验证（**你在购买的时候要确认一下**）



如果你使用的是代充，那么在使用CodeX的时候，出现了手机解码问题，你需要借助一些接码平台
