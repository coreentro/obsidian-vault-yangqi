## 文件夹分工

| 文件夹 | 用途 |
|---|---|
| `00-收件箱` | 暂时无法判断归属、还没整理完的材料 |
| `AI` | AI、编程、结构化语言、Agent、API 等知识卡片 |
| `工具` | 飞书、订阅、脚本命令等具体工具和操作记录 |
| `科研` | 课题、实验计划、科研工作、概念和心得 |
| `科研/文献笔记` | 科研文献的结构化阅读笔记 |
| `科研/附件PDF` | 科研文献的原始 PDF |
| `科研/PPT` | 组会汇报和课题演示文稿 |
| `本科生` | 本科生实验指导、实验步骤、教学材料 |
| `knoevenagel文献` | Knoevenagel 相关文献、概念卡片和实验条件 |
| `读书` | 书籍阅读笔记 |
| `Clippings` | 网页剪藏和外部材料 |
| `assets` | 图片、附件和素材 |

## 整理原则

1. 能明确用途的笔记，放进对应实践场景。
2. 暂时说不清的材料，先放 `00-收件箱/待整理`。
3. 附件统一留在 `assets`，正文笔记只保留引用。
4. 文件名尽量表达内容，不保留过多 `未命名`、缩写和临时名。
5. 后续如果某一类材料变多，再单独分化出新的文件夹。
